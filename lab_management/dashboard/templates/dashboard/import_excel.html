Expected Excel Format

The system can import both clusters and nodes from the same Excel file. Rows with CLUSTER NAME but no SERVICE TAG are treated as cluster definitions. Rows with SERVICE TAG are treated as node entries.

Cluster Columns (Optional)

Column Name Required Description
CLUSTER NAME Yes (for clusters) Unique cluster name
USER No Cluster user name
DESCRIPTION No Cluster description
CLUSTER TYPE No Cluster category (infra/provision/validation)

Server Columns

Column Name Required Description
SERVICE TAG Yes (for nodes) Unique service tag identifier
ROLE No Server role (Master, Worker, Storage, GPU, Network)
SERVER MODEL No Server model name/number
GENERATION No Server generation
CLUSTER NAME No Assign server to specific cluster
IDRAC IP No iDRAC IP address
IDRAC CREDS No Format: username:password
BMC MAC ADDRESS No BMC MAC address
PXE MAC ADDRESS No PXE boot MAC address
